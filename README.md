# node-powerfx

Example repository to demonstrate how to call PowerFx

## Getting started

1. Ensure .Net 8.0 SDK installed

2. Change to PowerFx module

```bash
cd src/PowerFx
```

3. Publish the example Power Fx class

```bash
dotnet publish
```

4. Change to Node JS code

```
cd ..
npm install
```

5. Run the app

```
node index.js
```
